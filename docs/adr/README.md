# Architecture Decision Records

## O que são ADRs?

Architecture Decision Records (ADRs) documentam decisões arquiteturais importantes tomadas durante o desenvolvimento do sistema.

Cada ADR descreve:

- Contexto
- Problema
- Decisão
- Consequências
- Alternativas consideradas

Este projeto segue o formato proposto por Michael Nygard.
