# Appointments

Gerenciamento dos agendamentos.

---

## Agendar

POST

```
/api/appointments
```

---

## Buscar

GET

```
/api/appointments/{id}
```

---

## Listar

GET

```
/api/appointments
```

---

## Atualizar

PUT

```
/api/appointments/{id}
```

---

## Cancelar

PATCH

```
/api/appointments/{id}/cancel
```

---

## Confirmar

PATCH

```
/api/appointments/{id}/confirm
```

---

## Finalizar

PATCH

```
/api/appointments/{id}/complete
```
