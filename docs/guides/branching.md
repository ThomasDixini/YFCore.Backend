# Estratégia de Branches

## Main

Contém apenas versões estáveis.

---

## Develop

Integra funcionalidades antes da publicação.

---

## Feature

Novas funcionalidades.

Exemplo:

```
feature/create-appointment
```

---

## Fix

Correções.

Exemplo:

```
fix/appointment-validation
```

---

## Hotfix

Correções críticas em produção.

Exemplo:

```
hotfix/login-error
```

---

# Fluxo

```
feature

↓

develop

↓

main
```
